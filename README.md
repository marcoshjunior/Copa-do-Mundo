# Simulador de Copa do Mundo

Aplicação web que simula uma Copa do Mundo completa, desde a fase de grupos até a final, utilizando dados reais de seleções via API.



## Funcionalidades

* Sorteio automático dos grupos (A até H)
* Simulação da fase de grupos (todos contra todos)
* Classificação com base em pontos e saldo de gols
* Mata-mata com pênaltis em caso de empate
* Exibição de:

  * Grupos
  * Jogos da fase de grupos
  * Tabela com pontuação
  * Chaveamento do mata-mata
* Integração com API externa
